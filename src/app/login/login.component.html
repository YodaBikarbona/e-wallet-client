Register

All fields on register form are required! Create your account using your information. Pick your country and your city (not possible to change), and set valid email (get your login code on mail)

Login

Use your email and password to login, first login required login code (activation code) you received on email. User can resend code if not got on mail.

Dashboard

News

There will be all news (whats new added or changed on application)

Profile

On tab profile user can see personal information, application information, change a password (after changing user account will be deactivated and will get login code again on email). Through Edit profile form user can change personal information, and set default currency (only from activated user currencies, it helps to add bills without picking currency on new bill form, if user doesn't have default currency then user needs pick currency on new bill form).

Bills

On this tab user can see costs and profits. Through button switch user can switch type of bills. Report (PDF) button will make report and download on your device. Filters (category, subcategory, currency) help to get more precise results. On table user can change limit and offset of results (max 20 results on one page, pdf report will use all results that user got using filters). User can see bill details, clicking on button Show details. Through button New bill user can add new bill using category (required), subcategory, currency (required), title (this column is recommended and required for article name e.g. category "Food" subcategory "Meat" title "Chicken") comment (e.g. I brought a new smartphone in Mobile shop), price (required), quantity (required, cannot be less than 0), Not my City (all bills that not made in your city e.g. if user live in London and he brought new smartphone in Paris, it helps for new feature for users who want make some research to compare user city prices with some another city using specific category or subcategory e.g. if user wants to see how many people spend money for Fuel in London and Paris, where is bigger price for some category or subcategory and how much. In some period is "Fuel" get cheaper or more expensive)

Graph

On this tab user can see graph of costs and profits (their own), also can see monthly limit, are costs are close to limit, see min and max cost and profit

Options

User can set only categories, subcategories and currencies (max 10) they want to use. This is first step if user wants add new bills, filters will show only categories, subcategories and currencies that user pick (no need to see all categories and wasting time to find your category). User also can set their own limits for some currency (next feature will send warnings if user is to close to monthly limit)

Suggestions

Not implemented yet! User can send message to moderators to add new category, subcategory, city (only moderator can change city) they want to use. If on register form user can't pick city that wants, pick any other and we will change currently city and country.

Research and analyze

Not implemented yet! User can make they own research to compare prices and how users spend money for specific category or subcategory between two cities
